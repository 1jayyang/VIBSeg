# VIBSeg

**VIBSeg: Filtering Task-Irrelevant Textual Semantics for Reliable Text-Guided Semi-Supervised Medical Image Segmentation**

🚧 The code is currently being organized and will be released soon.

VIBSeg addresses the problem of task-irrelevant semantics in pretrained text embeddings for text-guided semi-supervised medical image segmentation. It introduces a Text Variational Information Bottleneck (TextVIB) for semantic filtering and a cross-modal alignment mechanism (TCA) to improve foreground-background discriminability.
